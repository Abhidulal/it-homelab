# IT Home Lab

A collection of hands-on IT labs documenting real-world system administration skills built from scratch.

## Environment

| Component | Details |
|---|---|
| Host Machine | MacBook Air M3 (8GB RAM) |
| Virtualisation | UTM (ARM emulation) |
| Server OS | Windows Server 2022 Standard Evaluation |

## Labs
| Lab | Topic |
|---|---|
| [Lab 1](windows-server/01-dhcp-dns-fileserver.md) | DHCP, DNS, and File Server configuration |
| [Lab 2](windows-server/02-active-directory.md) | Active Directory, Domain Controller, Users and Groups |
| [Lab 3](windows-server/03-group-policy.md) | Group Policy Object (GPO) configuration |
| [Lab 4](windows-server/04-domain-administration.md) | Domain Administration, User Lifecycle, Password and Lockout Policies |

## Skills Covered

- Windows Server 2022 installation and configuration
- DHCP scope creation and management
- DNS zone and A record configuration
- Active Directory Domain Services
- Promoting a server to Domain Controller
- Organisational Units, Users, and Security Groups
- Group Policy Object creation and linking
- Command line tools: nslookup, gpupdate, gpresult, ipconfig
